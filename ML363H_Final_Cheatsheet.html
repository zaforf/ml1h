1. Mistake-Bounded (MB) Model

1.1 Definition

  • Mistake bound $M$ for $\mathcal{C}$: for every target $f\in\mathcal{C}$ and every labeled sequence consistent with $f$, the learner makes at most $M$ mistakes.

1.2 Elimination (Monotone / General Disjunctions)

Monotone disjunctions: $f(x)=x_{i_1}\lor\cdots\lor x_{i_k}$ over $\{0,1\}^d$.

  • Algorithm (sketch): init $h = x_1\lor\cdots\lor x_d$. On $(x,y)$, predict $\hat{y}=h(x)$; on false positive, delete every $x_i$ with $x_i=1$.
  • Invariant: any literal that appears in the true target is never removed.
  • Bound: start with $d$ literals, each false-positive removes $\ge1$ literal, so $\#\text{mistakes}\le d$.
  • Takeaway: elimination is the canonical example of a simple MB algorithm: mistakes correspond to ‚Äúruling out‚Äù wrong literals.

General disjunctions (with negations): introduce $Y_i=\bar{X}_i$, run elimination on $2d$ literals ‚áí $MB\le 2d$.

1.3 Halving Algorithm (Finite $\mathcal{C}$)

  • Maintain version space $V_t = \{h\in\mathcal{C}$ consistent with seen examples$\}$.
  • Predict by majority vote over $V_t$; on a mistake, remove all hypotheses agreeing with the wrong prediction.
  • Key inequality: after each mistake, $|V_{t+1}|\le|V_t|/2$.
  • Thus after $M$ mistakes, $|V_M|\le|\mathcal{C}|/2^M$; but must have $|V_M|\ge1$ (true $f$). So $2^M\le|\mathcal{C}|$, i.e. $$M\le\log_2|\mathcal{C}|.$$
  • Takeaway: ‚Äúeach mistake halves the version space‚Äù $\Rightarrow$ information-theoretically optimal MB $\approx \log_2|\mathcal{C}|$.

1.4 Decision Lists (Orders of Rules)

A 1-decision list is an ordered if-then-else over literals: $(L_1\to b_1),\dots,(L_k\to b_k)$, default $b_0$.

  • 1-decision lists (single-literal $L_i$) of length $k$ on $n$ vars: $MB = O(nk)$.
  • $t$-decision lists (conjunctions of $\le t$ literals per rule): $MB = n^{O(t)}$.

1B. Proof templates

  • Finite-class PAC: bad $h$ has $\Pr[\text{consistent on }m]\le(1-\epsilon)^m$; union over $\mathcal{C}$; pick $m\ge\frac1\epsilon\ln\frac{|\mathcal{C}|}{\delta}$.
  • Potential / weights: bound $W_{t+1}$ vs $W_t$ (Winnow, WM, boosting); sandwich total weight vs weight of mistakes or best expert.
  • Margin: grow $w_t\cdot w^*$, bound $\|w_t\|$, finish with Cauchy‚ÄìSchwarz (standard and SQ perceptron variants).

2. Multiplicative Updates & Experts

2.1 Winnow

Sparse disjunction target with $k\ll d$ relevant variables.

  • Maintain $w_i\ge0$, init $w_i=1$, threshold $\theta=d$; predict $1$ if $\sum_i w_i x_i\ge\theta$.
  • Type A (false negative): $y=1,\hat{y}=0$. For all $i$ with $x_i=1$, set $w_i\leftarrow 2w_i$.
  • Type B (false positive): $y=0,\hat{y}=1$. For all $i$ with $x_i=1$, set $w_i\leftarrow 0$ (or scale down).

Potential: $W = \sum_i w_i$.

  • Each relevant variable can be doubled at most $\log_2 d$ times before its weight alone would exceed $\theta=d$ ‚áí at most $k\log_2 d$ Type-A mistakes.
  • On Type A: $W$ increases by $
  • On Type B: $W$ decreases by $\ge d$ (since $\sum_i w_i x_i\ge\theta=d$).
  • $W$ starts at $d$ and never negative ‚áí $\#$Type B mistakes $\le$ const¬∑$\#$Type A mistakes.
  • Bound: $MB \le 2k\log_2 d = O(k\log d)$.
  • Takeaway: when the target is sparse ($k\ll d$), Winnow‚Äôs MB depends on $k$ not $d$ ‚áí attribute-efficient learning.

2.2 Weighted Majority (Deterministic)

  • $K$ experts, weights $w_i^{(t)}$ (init 1); predict by weighted majority. On mistake: halve weights of wrong experts.

Potential: $W_t=\sum_i w_i^{(t)}$.

  • On a mistake, at least half of $W_t$ was wrong; after halving them, $$W_{t+1}\le \tfrac{1}{2}\cdot\tfrac{W_t}{2}+\tfrac{W_t}{2}=\tfrac{3}{4}W_t.$$
  • Thus after $M$ mistakes: $W_M\le K(3/4)^M$.
  • Best expert with $m$ mistakes has weight $(1/2)^m$ ‚áí $(1/2)^m\le W_M$.
  • Combine: $$(1/2)^m\le K(3/4)^M \Rightarrow M\le \frac{\ln2}{\ln(4/3)}\bigl(m+\log_2K\bigr)\approx 2.4(m+\log_2K).$$
  • Takeaway: we compete with the best expert; our mistakes are within a constant factor plus a $\log K$ overhead.

2.3 Randomized Weighted Majority (RWM)

  • Parameter $\epsilon\in(0,1/2)$. If expert $i$ is wrong: $w_i\leftarrow(1-\epsilon)w_i$.
  • Prediction: choose expert $i$ with prob. $w_i/W_t$; let $\ell_t$ be expected loss (weighted fraction wrong).

Weight recurrence: $W_{t+1}=W_t(1-\epsilon\ell_t)$.

Bound idea:

  • Best expert with $m$ mistakes has weight $(1-\epsilon)^m$ ‚áí $(1-\epsilon)^m\le K\prod_t(1-\epsilon\ell_t)$. Take logs and use $-x^2-x\le\ln(1-x)\le -x$ for $x\in(0,1/2)$: $$\sum_t \ell_t \le (1+\epsilon)m + \frac{\ln K}{\epsilon}.$$
  • So $\mathbb{E}[M]\le (1+\epsilon)m + \frac{\ln K}{\epsilon}$.
  • Takeaway: by randomizing, we can get an expected loss arbitrarily close to $(1+\epsilon)$ times the best expert, trading off with an additive $\ln K/\epsilon$ term.

3. PAC Learning & Sample Complexity

3.1 Finite Classes, Realizable Case

  • Bad $h$: $\mathrm{err}_D(h)>\epsilon$.
  • For fixed bad $h$: $\Pr[\text{consistent with $m$}] \le (1-\epsilon)^m \le e^{-\epsilon m}$.
  • Union bound over $\mathcal{C}$: $$\Pr[\exists\ \text{bad $h$ consistent}] \le |\mathcal{C}|e^{-\epsilon m}.$$
  • To make this $\le\delta$: $$m \ge \frac{1}{\epsilon}\ln\frac{|\mathcal{C}|}{\delta}.$$
  • Any consistent learner is $(\epsilon,\delta)$-PAC.
  • Takeaway: finite-class PAC: more hypotheses and smaller $\epsilon,\delta$ mean more samples, via counting + union bound.

3.2 Axis-Parallel Rectangles

  • Algorithm: tightest-fitting axis-aligned rectangle around positives; define 4 strips (top/bottom/left/right) of mass $\epsilon/4$; error $>\epsilon$ implies some strip unseen.
  • Miss-prob. for a strip: $(1-\epsilon/4)^m$; by union bound: $$\Pr[\text{failure}] \le 4(1-\epsilon/4)^m \le 4 e^{-\epsilon m/4}.$$
  • Set $\le\delta$ ‚áí $$m \ge \frac{4}{\epsilon}\ln\frac{4}{\delta}.$$

3.3 MB $\Rightarrow$ PAC (Idea)

  • If a class is MB-learnable with bound $M$, it is PAC-learnable with roughly $$m = O\!\left(\frac{1}{\epsilon}\bigl(M + \log\tfrac{1}{\delta}\bigr)\right).$$
  • Sketch: MB bound $M$ ‚áí on $m$ fresh examples, at most $M$ are misclassified. Use concentration on empirical vs true error plus a union/conditioning argument.
  • Takeaway: an online MB guarantee can be converted into a distributional PAC guarantee with only a $1/\epsilon$ and $\log(1/\delta)$ overhead.

3.4 Empirical vs true error (uniform convergence)

  • For a fixed $h$, Hoeffding on a hold-out set of size $n$ gives $$\Pr\bigl[|\widehat{\mathrm{err}}(h) - \mathrm{err}_D(h)| \ge \epsilon\bigr] \le 2e^{-2n\epsilon^2}.$$
  • So it is enough to take $$n \ge \frac{1}{2\epsilon^2}\ln\frac{2}{\delta}$$ to ensure $|\widehat{\mathrm{err}}(h) - \mathrm{err}_D(h)| \le \epsilon$ with prob. $\ge 1-\delta$.
  • Takeaway: empirical error on a held-out subset is close to true error once you have $\Theta(1/\epsilon^2)$ many hold-out points.

4. VC Dimension & Radon

4.1 Definitions

  • Shattering: $\mathcal{C}$ shatters $S$ if for every labeling of $S$ there is $h\in\mathcal{C}$ realizing it.
  • VC-dim: $\mathrm{VC}(\mathcal{C})$ = max $|S|$ that can be shattered.
  • Examples: intervals (VC=2), axis-parallel rectangles in $\mathbb{R}^d$ (VC=$2d$), halfspaces in $\mathbb{R}^d$ (VC=$d+1$).

4.2 Fundamental Theorem (VC version)

  • Uniform convergence (matches notes): Draw $m = O((\mathrm{VC}(\mathcal{C}) + \log(1/\delta))/\epsilon^2)$ samples. With prob. $\ge 1-\delta$: $|\widehat{\mathrm{err}}(h) - \mathrm{err}(h)| \le \epsilon$ for all $h \in \mathcal{C}$. Empirical error approximates true error for every $h\in\mathcal{C}$.
  • Realizable variant: consistent learner needs only $m = O((d + \log(1/\delta))/\epsilon)$ to get $\mathrm{err}_D(h)\le\epsilon$ (no $\epsilon^2$).
  • Takeaway: VC-dim is the capacity measure; $\epsilon^2$ when bound holds uniformly over all $h\in\mathcal{C}$.

4.3 Radon & VC(Halfspaces)

  • Radon‚Äôs theorem: any $d+2$ points in $\mathbb{R}^d$ can be partitioned into $A,B$ with $\mathrm{conv}(A)\cap\mathrm{conv}(B)\ne\emptyset$ (via an affine dependence $\sum\alpha_i x_i=0$, $\sum\alpha_i=0$).
  • Label $A$ as $+1$, $B$ as $-1$; the common point of the hulls would have to be both $+1$ and $-1$, so no halfspace can realize this labeling ‚áí no set of $d+2$ points is shattered. Together with a $d+1$-point construction, VC-dim of halfspaces is $d+1$.

5. Regression & Optimization

5.1 Regression Basics

  • Bayes regressor: for squared loss, $g^*(x)=\mathbb{E}[Y|X=x]$ minimizes $\mathbb{E}[(Y-g(X))^2]$.
  • OLS (matrix): minimize $\|Xw-y\|^2$; gradient $2X^T(Xw-y)$; normal equations $X^TXw = X^Ty$ ‚áí $w = (X^TX)^{-1}X^Ty$.
  • Simple linear regression $y = \beta_0 + \beta_1 x$: $$\beta_1 = \frac{\mathrm{cov}(x,y)}{\mathrm{var}(x)}, \qquad \beta_0 = \bar{y} - \beta_1\bar{x}.$$
  • Ridge: minimize $\|Xw-y\|^2 + \lambda\|w\|^2$ ‚áí $w = (X^TX+\lambda I)^{-1}X^Ty$.

5.2 Bias‚ÄìVariance (scalar form + Ridge formulas)

  • For any estimator $\hat{a}$ of $a$: $$\mathrm{MSE}(\hat{a}) = \mathbb{E}[(\hat{a}-a)^2] = \underbrace{\mathrm{Var}(\hat{a})}_{\text{variance}} + \underbrace{(\mathbb{E}[\hat{a}]-a)^2}_{\text{bias}^2}.$$
  • Ridge toy model $y^i = a^* x^i + \epsilon^i$, $\epsilon^i\sim N(0,\sigma^2)$, $\beta = \frac{1}{m}\sum_i (x^i)^2$: $$\text{Variance} = \frac{m\beta\sigma^2}{(\lambda + m\beta)^2}, \qquad \text{Bias}^2 = \frac{\lambda^2 (a^*)^2}{(\lambda + m\beta)^2}.$$ So $\mathrm{MSE} = \text{Variance} + \text{Bias}^2$.
  • When is variance minimized? Variance is large at $\lambda=0$ (OLS) and decreases as $\lambda$ increases (eventually ‚Üí 0 as $\lambda\to\infty$). So variance is minimized in the limit $\lambda\to\infty$ (all weight shrunk). But MSE is minimized at a finite $\lambda$ because bias¬≤ grows with $\lambda$.
  • Optimal $\lambda$ (minimizes MSE in this toy model): $$\lambda^* = \frac{\sigma^2}{(a^*)^2}.$$ So Ridge can beat OLS ($\lambda=0$) when there is noise ($\sigma^2>0$): we trade a little bias for lower variance.

5.3 GD, SGD, & Smoothness

  • GD update: $w_{t+1} = w_t - \eta \nabla f(w_t)$.
  • SGD: for $f(w)=\frac{1}{m}\sum_i f_i(w)$, use $w_{t+1} = w_t - \eta \nabla f_{i_t}(w_t)$ with random $i_t$.
  • Sigmoid: $\sigma(z) = \frac{1}{1+e^{-z}}$, $\sigma'(z)=\sigma(z)(1-\sigma(z))$.

$L$-smoothness: $\|\nabla f(x)-\nabla f(y)\|\le L\|x-y\|$ (gradient does not change too abruptly).

  • Descent Lemma: with $\eta=1/L$, $$f(x_{k+1}) \le f(x_k) - \frac{1}{2L}\|\nabla f(x_k)\|^2.$$
  • Summing: $$\sum_{k=1}^T \|\nabla f(x_k)\|^2 \le 2L(f(x_1)-f(x^*)).$$
  • Thus $$\min_{1\le k\le T}\|\nabla f(x_k)\|^2 \le \frac{2L(f(x_1)-f(x^*))}{T}.$$
  • Takeaway: for $L$-smooth $f$, the smallest gradient norm encountered after $T$ steps is at most $O(1/\sqrt{T})$ in norm. A small gradient means that $f$ is locally flat around that iterate; for convex $f$ this corresponds to being close to a minimizer.

6. Perceptron & Losses

6.1 Perceptron Margin Bound (full sketch)

  • We can assume $\|x_i\|\le 1$ (scale inputs) and there is a unit vector $w^*$ such that for all $i$, $$y_i (w^*\cdot x_i) \ge \rho > 0.$$ Here $\rho$ is the geometric margin.
  • Perceptron update on a mistake $(x_t,y_t)$: $w_{t+1} = w_t + y_t x_t$, with $w_0 = 0$.
  • Alignment with $w^*$: after one mistake, $$w_{t+1}\cdot w^* = w_t\cdot w^* + y_t (x_t\cdot w^*) \ge w_t\cdot w^* + \rho.$$ So after $M$ mistakes: $$w_M\cdot w^* \ge M\rho.$$
  • Norm growth: on a mistake, $$\|w_{t+1}\|^2 = \|w_t + y_t x_t\|^2 = \|w_t\|^2 + 2y_t(w_t\cdot x_t) + \|x_t\|^2.$$ Because we erred, $y_t (w_t\cdot x_t) \le 0$, so $$\|w_{t+1}\|^2 \le \|w_t\|^2 + \|x_t\|^2 \le \|w_t\|^2 + 1.$$ Thus $\|w_M\|^2 \le M$ and $\|w_M\|\le\sqrt{M}$.
  • Cauchy‚ÄìSchwarz: $w_M\cdot w^* \le \|w_M\|\|w^*\| = \|w_M\|$. Together with the previous inequalities: $$M\rho \le w_M\cdot w^* \le \|w_M\| \le \sqrt{M}.$$ Hence $\sqrt{M} \le 1/\rho$ ‚áí $M \le 1/\rho^2$.
  • Takeaway: if the data are well separated with margin $\rho$, the perceptron makes at most $1/\rho^2$ mistakes; increasing the margin directly reduces the number of updates.

6.2 Losses & Convexity

  • Square loss: $(y-w^Tx)^2$. Natural for regression; gradient is $-2(y-w^Tx)x$.
  • Hinge loss: $\max(0,1-yw^Tx)$. Zero whenever $y w^Tx \ge 1$; penalizes points inside the margin or misclassified. SGD on hinge with update only when $1-yw^Tx>0$ closely matches perceptron updates.
  • Logistic loss: $\ln(1+e^{-yw^Tx})$. Smooth and always positive; corresponds to the negative log-likelihood under a logistic model, and leads to probabilistic outputs via $\sigma(w^Tx)$.
  • Convexity (first-order): $f$ convex iff $$f(y) \ge f(x) + \nabla f(x)^T(y-x)\ \ \forall x,y.$$

7. Boosting & Weak Learners

7.1 Setup

  • $N$ training points, $T$ rounds. Weak learner: each $h_t$ has weighted error $\epsilon_t = \mathrm{err}(h_t) \le \frac12 - \gamma$ (edge $\gamma>0$).
  • $\beta$-rule (notes / Schapire-style): $\beta = \frac{\frac12-\gamma}{\frac12+\gamma}\in(0,1)$. One common update: multiply weight of correct points by $\beta$, leave incorrect weights unchanged; init $w_i=1/N$, total $W^{(1)}=1$.
  • Round $t$ uses only $h_t$: the weight vector at step $t$ is renormalized from updates by the latest weak hypothesis; the ensemble vote is separate‚Äîpast mistakes are ‚Äúremembered‚Äù indirectly through the current distribution.
  • Final classifier: majority vote of $h_1,\dots,h_T$.

7.2 Per-round total weight (upper bound route)

  • Let $\epsilon_t$ be the weighted error of $h_t$. One step: $$W^{(t+1)} = W^{(t)}\bigl((1-\epsilon_t)\beta + \epsilon_t\bigr).$$
  • Worst case $\epsilon_t = \frac12-\gamma$ gives $$W^{(t+1)} \le W^{(t)}(1-2\gamma).$$ Hence $W^{(T+1)} \le (1-2\gamma)^T$.

7.3 Lower bound via misclassified points

  • If the final majority misclassifies $x_i$, then $h_t$ was correct on $x_i$ for at most $T/2$ rounds, so $w_i$ was multiplied by $\beta$ at most $T/2$ times: $$w_i^{(T+1)} \ge \frac1N \beta^{T/2}.$$
  • With training error rate $E_{\mathrm{train}}$ (fraction wrong), $$W^{(T+1)} \ge E_{\mathrm{train}}\,\beta^{T/2}.$$

7.4 Combine (exponential decay)

  • Sandwich: $E_{\mathrm{train}}\beta^{T/2} \le W^{(T+1)} \le (1-2\gamma)^T$ ‚áí $$E_{\mathrm{train}} \le \left(\frac{1-2\gamma}{\sqrt\beta}\right)^T = (1-4\gamma^2)^{T/2} \le e^{-2\gamma^2 T}$$ using $1-x\le e^{-x}$.
  • Rounds / sample complexity story: $T = \Theta\!\left(\frac{1}{\gamma^2}\log\frac{1}{\epsilon}\right)$ merges exponential decay in $T$ (from the weight sandwich) with the need for enough samples per round so the empirical weak learner really has edge $\gamma$ (variance $\sim 1/\gamma^2$ via Chernoff/Hoeffding-style arguments).
  • Balanced weak learner (branching): need edge on both conditionals $D_+,D_-$ (e.g. accuracy $\ge \frac12+\gamma$ on each). A rule that only predicts majority class globally can look ‚Äúweak‚Äù on one side and give no progress when data are split.
  • Branching-program analysis: total weight decay $W_T \le W(1-2\gamma)^T$; martingale / Azuma controls deviations in the random-walk view of accumulated votes.

8. Confidence Intervals & Comparing Classifiers

8.1 Single proportion

  • $Y_i = \mathbf{1}[\text{$h$ correct on $x_i$}]$, $\hat p = \frac1n\sum_i Y_i$, $\mathbb E[\hat p]=p$.
  • $\mathrm{Var}(Y_i)=p(1-p)$, $\mathrm{Var}(\hat p)=\frac{p(1-p)}{n}$, $\mathrm{SE}(\hat p)=\sqrt{\frac{p(1-p)}{n}}$.
  • CLT: $Z=\frac{\hat p - p}{\mathrm{SE}(\hat p)} \approx N(0,1)$ for large $n$.
  • 95% CI: $p \in \hat p \pm 1.96\cdot \widehat{\mathrm{SE}}$ (notes often substitute $2$).
  • 80% CI: $z\approx 1.28$ (notes ~1.3).

8.2 Difference of two error rates

  • $\widehat\Delta = \hat e_1 - \hat e_2$; if independent samples, $$\mathrm{Var}(\widehat\Delta) \approx \frac{\hat e_1(1-\hat e_1)}{n_1} + \frac{\hat e_2(1-\hat e_2)}{n_2}.$$
  • Build CI for $\Delta$ the same way using normal approximation.

9. Random Projections & Johnson‚ÄìLindenstrauss

9.1 One Gaussian projection

  • Draw $r\sim N(0,I_d)$, $f_r(x)=\langle x,r\rangle$. For $\Delta=x-y$: $\mathbb E[f_r(x)-f_r(y)]=0$.
  • $\mathrm{Var}(f_r(x)-f_r(y))=\|\Delta\|_2^2$ and $\mathbb E[(f_r(x)-f_r(y))^2]=\|x-y\|_2^2$.

9.2 JL lemma (what to remember)

  • Map $x\mapsto \frac{1}{\sqrt K}Ax$ with $A\in\mathbb R^{K\times d}$ i.i.d. Gaussian (or subgaussian / sparse sign). For $m$ points, need $$K = O\!\left(\frac{\log m}{\epsilon^2}\right)$$ so all pairwise distances are preserved within $(1\pm\epsilon)$ w.h.p. (union bound over $\binom{m}{2}$ pairs + concentration of chi-square / subgaussian sums).
  • Curse of dimensionality tie-in: JL reduces effective dimension for distance-based methods (e.g. NN) before clustering or search.

10. PCA, SVD, Power Method

10.1 PCA objectives

  • 1-D: $\max_{\|v\|=1} \frac1m\sum_j \langle x^j,v\rangle^2 = \max_{\|v\|=1} v^T(X^TX)v$ (variance along $v$).
  • $k$-D: orthonormal $v_1,\dots,v_k$ maximizing projected variance $\frac1m\sum_j\sum_{i=1}^k \langle x^j,v_i\rangle^2$.
  • Data often whitened: subtract mean; optionally scale by std per feature.

10.2 Spectral facts

  • $X^TX$ is symmetric PSD ‚áí eigendecomposition $Q\Lambda Q^T$, eigenvalues $\lambda_i\ge0$.
  • First principal direction = eigenvector for $\lambda_{\max}$.

10.3 Power method (sketch)

  • Iterate $u_{t+1} = Au_t / \|Au_t\|$ for PSD $A$ (or $A=X^TX$). If $\lambda_1>\lambda_2$ and initial $u_0$ has nonzero component along top eigenvector, $u_t$ converges to $v_1$.
  • Rate: error shrinks like $(\lambda_2/\lambda_1)^t$ ‚áí to reach accuracy $\epsilon$ in direction, $$t = O\!\left(\frac{\log n + \log(1/\epsilon)}{\log(\lambda_1/\lambda_2)}\right).$$

10.4 SVD & least squares

  • $A = U\Sigma V^T = \sum_i \sigma_i u_i v_i^T$. Right singular vectors (columns of $V$): eigenvectors of $A^TA$ ‚Äî for data matrix $X$, these are the principal directions (axes of maximum variance). Truncated $V_k^T$ maps a row of $X$ to its $k$-D PCA coordinates.
  • Left singular vectors (columns of $U$): eigenvectors of $AA^T$. The matrix $U_k\Sigma_k$ gives scores / projected data in the $k$-D subspace; $U_k$ alone is the normalized version of those loadings.
  • $\sigma_i^2$ = eigenvalues of $A^TA$. For centered sample matrix $X\in\mathbb R^{m\times d}$, sample variance along $v_i$ ties to $\sigma_i^2/(m-1)$ (same directions as PCA on covariance $\propto X^TX$).
  • Condition number: $\kappa(X^TX)=\kappa(X)^2$ ‚Äî forming normal equations squares singular values, so GD on least squares can slow badly vs SVD/pseudoinverse.
  • PCA vs rank-$k$ SVD: PCA maximizes projected variance; truncating SVD minimizes Frobenius reconstruction error ‚Äî for centered data, same subspace (Eckart‚ÄìYoung / Pythagorean decomposition).
  • Rank-$k$ approximation (Eckart‚ÄìYoung): $A_k=\sum_{i=1}^k \sigma_i u_i v_i^T$ minimizes $\|A-B\|_F$ among $\mathrm{rank}(B)\le k$.
  • Least squares: $\min_x\|Ax-b\|^2$ ‚áí $x = V\Sigma^\dagger U^T b$.
  • PCA from SVD: $X=U\Sigma V^T \Rightarrow X^TX = V\Sigma^T\Sigma V^T$.
  • Matrix completion (heuristic): impute missing entries $\to$ truncated SVD $\to$ iterate.

11. $k$-Means & Unsupervised Basics

  • Clusters $S_1,\dots,S_K$, centroids $\mu_i$. Objective: $$\min \sum_{i=1}^K \sum_{x\in S_i} \|x-\mu_i\|_2^2.$$
  • Lloyd: alternate assign point to nearest $\mu_i$, set $\mu_i$ to cluster mean (monotone decrease, local min).
  • Hardness: finding global optimum is NP-hard (notes).
  • Nearest neighbor reminder: 1-NN risk bounded in terms of Bayes risk but needs lots of data in high $d$; JL / feature reduction sometimes helps.

12. GLM / Logistic / Neural net hooks

  • GLM mean: $\mathbb E[Y|X]=u(w\cdot x)$ with monotone / Lipschitz link $u$.
  • Logistic: $\Pr[y=1|x]=\sigma(w\cdot x)$, NLL = logistic loss, convex in $w$.
  • Shallow net: $f(x)=\sum_j a_j\,\sigma(v_j\cdot x)$; universal approximation for reasonable $\sigma$.
  • Backprop pattern: multivariate chain rule on computational graph; store activations, propagate $\partial L/\partial$ upstream.

13. LSI, Min-wise hashing, Jaccard (quick)

  • Jaccard for sets $A,B$: $J=|A\cap B|/|A\cup B|$. Min-wise hashing: random permutation $\Rightarrow$ collision prob $\approx J$ (used for near-duplicate / similarity search).
  • LSI: term-document matrix $A$; $A^TA$ (doc‚Äìdoc) vs $AA^T$ (term‚Äìterm); truncated SVD gives latent topics; query scored vs rows via cosine similarity.

14. Statistical Queries, Parity, Hardness

14.1 SQ model

  • Learner chooses $\phi(x,y)\in[-1,1]$; oracle returns $q$ with $|q-\mathbb E[\phi(x,C(x))]|\le\tau$ (tolerance $\tau$).
  • Simulate one query from samples: Hoeffding $\Rightarrow$ $m = O(\frac{1}{\tau^2}\log\frac{1}{\delta})$; for $Q$ queries, union bound $\Rightarrow$ $\log(Q/\delta)$.
  • SQ-learnable $\Rightarrow$ PAC-learnable (with sample blow-up).

14.2 RCN / noise

  • Flipped label $\tilde y = zy$, $\mathbb E[z]=1-2\eta$. Then $\mathbb E[\phi(x,\tilde y)]=(1-2\eta)\mathbb E[\phi(x,y)]$; divide by $1-2\eta$, budget tolerance $\tau'=(1-2\eta)\tau$.

14.3 Monotone conjunctions and $\tau$

  • Query $\phi_i(x,y)=\mathbf{1}[x_i=0\wedge y=1]$. Oracle can be wrong by $\tau$ on each estimate; a bad literal kept because the answer was underestimated by $\tau$ can contribute up to $2\tau$ true error. Union over $n$ literals ‚áí use $$\tau \le \frac{\epsilon}{2n}$$ to guarantee total error $\le\epsilon$.

14.4 SQ Perceptron (noisy conditional mean)

  • Data from $D$, $\|x\|_2\le R$, separable by unit $w^*$ with margin $y(w^{*T}x)\ge\gamma>0$. Exact mistake-direction mean $v_t=\mathbb E[yx\mid y(w_t^Tx)<0]$ satisfies $w^{*T}v_t\ge\gamma$.
  • Oracle returns $\tilde v_t$ with per-coordinate error $\le\tau$ ‚áí $\|\tilde v_t-v_t\|_2\le \tau\sqrt d$. Update $w_{t+1}=w_t+\tilde v_t$.
  • Lower bound on progress: $w^{*T}\tilde v_t\ge w^{*T}v_t-\|\tilde v_t-v_t\|_2\ge \gamma-\tau\sqrt d$, so $w^{*T}w_t\ge t(\gamma-\tau\sqrt d)$.
  • Upper bound on norm: on mistakes $w_t^Tv_t<0$, so $w_t^T\tilde v_t\le \|w_t\|\tau\sqrt d$; and $\|\tilde v_t\|\le R+\tau\sqrt d$. Hence $\|w_{t+1}\|^2\le \|w_t\|^2+2\tau\sqrt d\|w_t\|+(R+\tau\sqrt d)^2$, which yields $\|w_t\|\le t\tau\sqrt d+\sqrt{tB}$ with $B=(R+\tau\sqrt d)^2$.
  • Combine with $w^{*T}w_t\le\|w_t\|$: need the linear coefficient in $t$ to stay positive when matched to the upper bound ‚áí $$\tau < \frac{\gamma}{2\sqrt d}$$ (necessary for the standard potential argument to close; if $\tau$ is too large, $\|w_t\|$ can grow too fast relative to $w^{*T}w_t$).
  • Coordinate query: $\phi_j(x,y)=y\,x_j\,\mathbf{1}[y(w^Tx)<0]$; estimate all $d$ coords to $\pm\tau$ ‚áí $m=O(\frac{1}{\tau^2}\log\frac{d}{\delta})$.

14.5 Parity / LPN / hardness

  • Parity with noise (LPN): SQ lower bounds and representation hardness.
  • SQ-dimension (idea): many nearly orthogonal concepts ‚áí many queries to distinguish.
  • 3-coloring $\to$ learning 3-term DNF: $+$ on vertices, $-$ on edges; consistent DNF encodes a coloring ‚áí NP-hard learning.

15. Intuition & facts (mostly not above)

Skip this if you already internalized the numbered sections; it avoids re-listing $\tau$ bounds, boosting sandwiches, JL $K$, PAC formulas, etc.

15.1 Which tail bound?

  • Markov ($X\ge0$): $\Pr[X\ge a]\le \mathbb E[X]/a$ ‚Äî only need a mean; usually a stepping stone.
  • Chebyshev: $\Pr[|X-\mu|\ge t]\le \mathrm{Var}(X)/t^2$ ‚Äî polynomial decay; good when you control variance but not boundedness.
  • Hoeffding (bounded i.i.d.): see ¬ß3.4 ‚Äî exponential decay in $n\Delta^2$; drives PAC counting, SQ simulation, CIs.
  • Azuma / bounded differences: martingale with bounded steps ‚áí Hoeffding-like tails without full independence ‚Äî branching boosting random-walk view (¬ß7).

15.2 Algebra habits

  • $1-x\le e^{-x}$ turns products $(1-\epsilon)^m$ into $e^{-\epsilon m}$ so you can solve for $m$ in one line.
  • Tighter $\ln(1-x)$ bounds (RWM): already in ¬ß2.3.

15.3 Union bound ‚Äî what you are ‚Äúpaying for‚Äù

  • $\Pr(\cup_i E_i)\le\sum_i\Pr(E_i)$: to make failure $\le\delta$, each rare event typically needs probability $\delta/(\text{#events})$.
  • That is why sample size often picks up $\ln|\mathcal{C}|$, $\ln(m^2)$ for JL pairs, or $\ln Q$ for $Q$ statistical queries ‚Äî not because each event is harder, but because there are many of them.

15.4 Why $\epsilon$ vs $\epsilon^2$ shows up

  • One fixed hypothesis on a hold-out: empirical error concentrates at rate $1/\sqrt n$ ‚áí to nail error within $\epsilon$ need $n\sim 1/\epsilon^2$ (¬ß3.4).
  • Uniform over a rich class: same ‚Äúvariance scale‚Äù theme via VC (¬ß4.2) ‚Äî empirical error simultaneously close for all $h$.
  • Realizable finite class: consistency is a product of per-example events, giving $(1-\epsilon)^m$ and often $m\sim 1/\epsilon$ instead (¬ß3.1).

15.5 Domain adaptation / shift (TDS notes)

  • Joint error $\lambda = \min_{c\in\mathcal{C}}\bigl(\mathrm{err}_{\mathrm{train}}(c)+\mathrm{err}_{\mathrm{test}}(c)\bigr)$: best compromise target across domains.
  • Test bound: $\mathrm{err}_{\mathrm{test}}(h)\le \mathrm{err}_{\mathrm{train}}(h)+\lambda+\mathrm{disc}(D_{\mathrm{train}},D_{\mathrm{test}})$ where discrepancy measures how much disagreement between classifiers in $\mathcal{C}$ shifts between domains.
  • Intuition: low train error is useless if (i) no good joint classifier exists ($\lambda$ large) or (ii) train and test disagree on which concepts separate points ($\mathrm{disc}$ large).

15.6 Projection / SQ ‚Äúshape‚Äù (pointers only)

  • JL: dimension $K$ scales with $\log(\text{#points})$ because you union-bound over pairs, not all functions ‚Äî details in ¬ß9.
  • SQ: per-coordinate noise $\tau$ blows up to $\ell_2$ noise $\Theta(\tau\sqrt d)$ when you stack coordinates ‚Äî that is why margin proofs need $\tau$ small in $d$; see ¬ß14.
